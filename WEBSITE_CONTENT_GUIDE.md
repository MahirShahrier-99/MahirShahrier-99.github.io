## Image Size Rules (applies everywhere)

| Use | Folder | Size |
|---|---|---|
| Grid card thumbnails (most sections) | `images/bg/` | **400 × 265 px** landscape — I'll resize if wrong |
| Grid card gallery/detail image (shown in popup) | `images/bg/` | Any size, landscape preferred |
| Work experience background photo | `images/bg/` | Any landscape photo |
| Carousel logo/icon (Scholarships, Awards, Funding, etc.) | `images/bg/` | Square or portrait, ~100×100 px ideal |
| Resources / Outreach thumbnails — book covers | `images/resources/` | Portrait |
| Resources / Outreach thumbnails — everything else | `images/resources/` | Landscape, ~120–300 px wide |

**Always tell me the exact filename you saved.** PDF gallery images are fine — I'll convert them.
